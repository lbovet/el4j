Cygwin convenience environment installation instructions

1. Copy .bash_profile to your home directory (C:\Documents and Settings\yourName\ on Windows or /home/yourName on Linux)
2. Copy bash_completion and the folder bash_completion.d to the etc folder of your cygwin installation (e.g. C:\cygwin\etc)
3. Open Console2, select Edit - Settings - Tabs - Shell and choose this cygwin.bat as shell.
