XML Schema versioning
=====================

Convention:
1. Version number of xml schema in attribute 'version'
2. Change history should be documented.
3. Current versions have normal filename like 'cookSwing.xsd'
4. Older versions should stay available: eg. 'cookSwing-1.3.xsd'
5. Version number should be consistent with version number of module version



For more information see http://www.xfront.com/Versioning.pdf
